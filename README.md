### Файлы с данными

Все данные находятся в директории `data/`:

- **`data/customer.csv`** - 4000 записей
  - Разделитель: точка с запятой (`;`)
  - Содержит информацию о клиентах: ID, имя, фамилия, дата рождения, профессия, адрес и т.д.

- **`data/product.csv`** - 190 записей
  - Разделитель: запятая (`,`)
  - Содержит информацию о продуктах: ID, бренд, линейка, класс, размер, цена и т.д.

- **`data/orders.csv`** - 20000 записей
  - Разделитель: запятая (`,`)
  - Содержит информацию о заказах: ID заказа, ID клиента, дата заказа, онлайн/офлайн, статус

- **`data/order_items.csv`** - 20000 записей
  - Разделитель: запятая (`,`)
  - Содержит информацию о составе заказов: ID позиции, ID заказа, ID продукта, количество, цены

## Использование

1. Запустите PostgreSQL: `make run`
2. Откройте ноутбук `hw2.ipynb` в Jupyter
3. Выполните ячейки по порядку:
   - Подключение к базе данных
   - Создание таблиц
   - Загрузка данных из CSV файлов
   - Выполнение SQL-запросов

## Примечания

⚠️ Исходные данные содержат ошибки (например, в `products` множество записей с `product_id=0`), поэтому данные загружаются AS IS, без установки Foreign Key.
